# Decisions log

Append-only record of decisions that shape this knowledge base and how the team works.
Newest at the bottom. Never edit past entries — if something changes, add a new entry
that supersedes the old one (reference its number).

Format per entry:

D-NNN — short title
- Who: <author slug>
- Decision: what was decided
- Why: the reasoning
- Consequences: what changes as a result
- Date: YYYY-MM-DD
